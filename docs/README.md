## About / Sobre

**EN:** Vali-Flow.Core targets `net8.0` and `net9.0`. The main entry point is `ValiFlow<T>` for in-memory and EF Core use; `ValiFlowQuery<T>` is the EF Core-safe variant. The library has zero NuGet dependencies.

**ES:** Vali-Flow.Core apunta a `net8.0` y `net9.0`. El punto de entrada principal es `ValiFlow<T>` para uso en memoria y con EF Core; `ValiFlowQuery<T>` es la variante segura para EF Core. La librería no tiene dependencias de NuGet.
